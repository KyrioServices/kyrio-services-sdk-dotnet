Determines cable providers that serve location specified by it's postal address. The method supports incomplete addresses: addressLine1 and postalCode or addressLine1, city and state.

Parameters
addressLine1Street number, pre-directional, street name, suffix, post-directional
addressLine2Secondary address line such as Apt, Suite or Lot
cityCity or town name
stateFor US addresses, usethe standard 2-character state abbreviation
postalCodeFor US addresses, use the 5-digit ZIP code
countryCodeUse ‘US’ to indicate US addresses. If the argument is omitted, ‘US’ will be assumed. Refer to ISO 3166 Country Code Standardfor non-US addresses.
Returns
Array of serviceability results from cable providers
